# 📦 Sistema Web de Controle de Estoque para Colégios

## 📌 Sobre o Projeto
Este projeto consiste em um **sistema web desenvolvido para modernizar e automatizar o processo de gestão de materiais de consumo** (como produtos de limpeza, higiene e materiais de escritório) em instituições de ensino.

A proposta do sistema é **substituir os métodos manuais de controle**, geralmente realizados em cadernos ou planilhas isoladas, que podem causar problemas como:

- perda de informações  
- falta de rastreabilidade  
- dificuldade de controle do estoque  
- uso excessivo de papel  

Com a digitalização do processo, o sistema permite **maior organização, transparência e eficiência na administração dos materiais escolares**.

---

## 🎯 Objetivos
O sistema foi desenvolvido com os seguintes objetivos principais:

- 📥 **Registrar entradas e saídas de materiais**, mantendo o estoque sempre atualizado.  
- 📊 **Automatizar o controle de estoque**, reduzindo erros humanos.  
- 🌱 **Diminuir o uso de papel**, tornando o processo mais sustentável.  
- 📑 **Gerar relatórios automáticos**, auxiliando gestores na tomada de decisões administrativas.

---

## 🚀 Funcionalidades Principais

O sistema oferece diversas funcionalidades para facilitar o gerenciamento de materiais:

- 📦 **Cadastro e gerenciamento de produtos**  
- 🏢 **Cadastro de setores da instituição**  
- 👤 **Gerenciamento de usuários do sistema**  
- 🔄 **Registro de movimentações de estoque** (entradas e saídas)  
- 🧾 **Rastreamento de solicitações de materiais** por usuário ou setor  
- 📊 **Geração de relatórios filtrados** por período, setor ou tipo de produto  
- ⚠️ **Monitoramento de níveis mínimos de estoque** com alertas automáticos  
- 🔐 **Controle de acesso por níveis de usuário**, como:
  - Gestor
  - Coordenador
  - Secretário
  - Agente administrativo

## 🛠️ Tecnologias Utilizadas

### Front-end
- **HTML5** — Estruturação das páginas da aplicação  
- **CSS3** — Estilização e design da interface  
- **JavaScript** — Interatividade e comunicação com a API  

### Back-end
- **Node.js** — Ambiente de execução JavaScript no servidor  
- **Express.js** — Framework para criação da API REST e gerenciamento de rotas  

### Banco de Dados
- **MySQL** — Sistema de gerenciamento de banco de dados relacional utilizado para armazenamento e integridade das informações

### Versionamento e Gestão do Projeto
- **Git & GitHub** — Controle de versão e hospedagem do repositório  
- **Trello** — Organização e gerenciamento das tarefas utilizando o método **Kanban**
